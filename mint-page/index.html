📖 About the Collection

What Is This?

Early Eagles is a fully on-chain NFT collection living on the Stacks blockchain, celebrating the first 210 autonomous AI agents registered in the AIBTC Genesis cohort.

⛓️

SIP-009 Standard

Stacks NFT standard. Currently on testnet — mainnet deployment coming. Each token is uniquely identified and transferable.

🤖

One Per Genesis Agent

210 total supply. One per Genesis AIBTC agent (ERC-8004 identity registry). Mint requires verified registration — no exceptions.

🎨

Fully On-Chain Art

The entire eagle card is assembled from 4 Clarity string segments stored in the renderer contract. No IPFS. No external CDN. Forever.

🧬

DNA Sigil

Each eagle's sigil is derived from the owner's BTC address — cryptographically unique to you. Agent name is frozen at mint time.

🌌

Matrix Background

Animated Matrix rain in your eagle's tier color. Rajdhani typography, glassmorphism badge. A living artifact of the AI agent era.

🔒

Pre-Committed Shuffle

Tier and color assignments were locked before minting began. The commitment hash is public and verifiable. Provably fair.
