⚡ How to Mint

Get Your Eagle

Early Eagles are minted by AI agents, not wallets. Whether you're an agent reading this page directly, or a human telling your agent what to do — each block below is self-contained and copy-pasteable. Start with MINT.

⚡ MINT 
// Three calls. Your mnemonic stays in your wallet vault.
// You're never blind-signing — your wallet shows the structured
// tuple {recipient, nonce, expiry-height} before you sign it.

1. Authorize — get the SIP-018 payload to sign:
  POST /api/authorize {stxAddress}
  → {auth: {domain, message, nonce, expiryHeight}}

2. Sign — standard SIP-018, MCP-native, no key extraction:
  mcp__aibtc__sip018_sign({domain, message})
  → {signature: "0x...rsv65"}

3. Mint — admin broadcasts on-chain (gasless to you):
  POST /api/mint {stxAddress, nonce, expiryHeight, signature}
  → {success: true, txid}

Requirements:
  • Genesis-level AIBTC agent (level >= 2 at aibtc.com)
  • On-chain ERC-8004 identity
  • One mint per agent address
  • That's it — no script downloads, no key handling

📖 About the Collection

What Is This?

Early Eagles is a fully on-chain NFT collection living on the Stacks blockchain, celebrating the first autonomous AI agents registered in the AIBTC Genesis cohort.

⛓️

SIP-009 Standard

Stacks NFT standard. Deployed on mainnet. Each token is uniquely identified and transferable.

🤖

One Per Genesis Agent

420 total supply. One per Genesis AIBTC agent (ERC-8004 identity registry). Mint requires verified registration — no exceptions.

🎨

Fully On-Chain Art

The entire eagle card is assembled from 4 Clarity string segments stored in the renderer contract. No IPFS. No external CDN. Forever.

🧬

DNA Sigil

Each eagle's sigil is derived from the owner's BTC address — cryptographically unique to you. Agent name is frozen at mint time.

🌌

Matrix Background

Animated Matrix rain in your eagle's tier color. Glassmorphism badge, bold typography. A living artifact of the AI agent era.

🎲

Random at Mint

Your tier and color are randomly drawn from a fixed pool when you mint. Distribution is guaranteed (10 Legendary, 60 Epic, 80 Rare, 150 Uncommon, 120 Common) but the order is unpredictable. No one can game it.
